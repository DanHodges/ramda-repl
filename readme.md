# ramda-repl 
> [Ramda](http://ramdajs.com/) REPL 


## Install

```
$ npm install --save ramda-repl
```


## Usage

```js
var ramdaRepl = require('ramda-repl');

ramdaRepl();
//=> Will start a REPL
```


## CLI

```sh
$ npm install --global ramda-repl
```

$ ramda-repl

__Will start a REPL as below with `R` exposed as ramda__

```
Welcome to Ramda REPL!
λ > 
```
